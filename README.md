# NeuroFlow Terminal

**NeuroFlow Terminal** — это интерактивная среда командной строки (REPL) для определения, построения и визуализации архитектур нейронных сетей с помощью доменно-специфического языка (DSL).

## Особенности

- **Интерактивный терминал**: Работайте с нейронными сетями в режиме реального времени
- **Синтаксическая подсветка**: Подсветка кода NeuroFlow DSL
- **Автодополнение**: Интеллектуальное автодополнение команд и функций
- **История команд**: Повторное использование предыдущих команд
- **Визуализация моделей**: ASCII-представление архитектуры сети
- **Управление сессией**: Сохранение и загрузка сессий

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd neuroflow-terminal
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

Запустите терминал:
```bash
python neuroflow_terminal/repl.py
```

## Язык NeuroFlow DSL

### Определение входных данных

```
input image [28, 28]
input labels [10]
```

### Определение модели

```
model SimpleNet {
  h1 = dense(image, 128) -> relu
  h2 = dense(h1, 64) -> relu
  output = dense(h2, 10) -> softmax
}
```

## Команды терминала

| Команда | Описание |
|--------|----------|
| `/help` | Показать справку |
| `/model` | Отобразить текущую модель |
| `/layers` | Показать статистику слоев |
| `/save <filename>` | Сохранить сессию |
| `/load <filename>` | Загрузить сессию |
| `/clear` | Очистить экран |
| `/exit` или `/quit` | Выйти из терминала |

## Архитектура проекта

```
neuroflow_terminal/

├── visualizer.py    # Визуализация моделей
├── completer.py     # Автодополнение

```

## Зависимости

- `prompt_toolkit` — для создания интерактивного терминала
- `pygments` — для синтаксической подсветки
- `torch` — для работы с нейронными сетями
- `numpy` — для числовых вычислений
